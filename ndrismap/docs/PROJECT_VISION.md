# NDRIS-Nepal
National Disaster, Responsibility & Impact System for Nepal

## Overview
NDRIS-Nepal is a civic-tech web platform that integrates disaster memory, citizen grievances, policy impact tracking, and an urban neglect index into a single interactive map-based system. The project is designed as a functional prototype for transparent governance and data-informed public accountability in Nepal.

## Core Objectives
- Preserve structured historical records of disasters in Nepal
- Enable citizens to submit grievances digitally
- Track policy implementation and perceived effectiveness
- Compute a district-wise Urban Neglect Index
- Visualize all data interactively on a Nepal map

## Systems Integrated
1. Disaster Memory System
2. Digital Citizen Grievance System
3. Policy Impact Tracking System
4. Urban Neglect Index Engine
5. Interactive Nepal Map (reference: INTmap)

## Target Users
- Citizens (public access)
- Administrators (data validation & analysis)

## Guiding Principles
- Neutral, non-political
- Data-driven, not opinion-driven
- Transparent, explainable logic
- Expandable beyond academic use
