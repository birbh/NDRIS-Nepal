# Development Rules for Copilot & Contributors

## Coding Rules
- Procedural PHP only
- MySQLi with prepared statements
- No frameworks (Laravel, React, etc.)
- Vanilla JavaScript only

## Security Rules
- Validate all inputs
- Prevent SQL injection
- No anonymous admin access

## Academic Rules
- Logic must be explainable in viva
- Avoid political language
- Use neutral terminology
- Comment all major logic blocks

## Commit Rules
- Small, descriptive commits
- No bulk “final” commits
- Meaningful commit messages

## Project Identity
This is a civic software prototype, not activism.
